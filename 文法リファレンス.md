# 文法リファレンス

コードを読み書きするときに参照する文法チートシート。

---

## プログラミング共通

### 変数の型
| 型 | 意味 | 例 |
|---|---|---|
| int / number | 整数・数値 | 1, 42, -5, 3.14 |
| string | 文字列 | "hello" |
| boolean | 真偽値 | true, false |
| null / undefined | 値がない | null |

### 演算子
```
算術:  +  -  *  /  %（余り = Mod）
比較:  ==  !=  <  >  <=  >=
論理:  &&（AND）  ||（OR）  !（NOT）
```

### 条件分岐
```
if (条件) { ... }
else if (別の条件) { ... }
else { ... }
```

### ループ
```
for (i = 0; i < 5; i++) { ... }   // 回数指定
while (条件) { ... }               // 条件が真の間
for (item of items) { ... }        // 配列の各要素
```

### 関数
```
function 名前(引数) {     // 引数 = 材料
    return 戻り値;        // 戻り値 = 料理
}
```

## TypeScript

### 変数宣言
```typescript
const name = "太郎";    // 変更できない（基本これ）
let age = 25;           // 変更できる
// var は古い書き方。使わない
```

### 型の指定
```typescript
const name: string = "太郎";
const age: number = 25;
const ok: boolean = true;
// 右辺で明らかなら型は省略OK
```

### アロー関数
```typescript
// 普通の関数
function add(a: number, b: number): number {
    return a + b;
}

// アロー関数（同じ意味）
const add = (a: number, b: number) => a + b;

// filter等で使う短い形
c => c.id === "123"
// ↑ (c) => { return c.id === "123"; } の省略
```

### interface（オブジェクトの形を定義）
```typescript
interface Todo {
    id: string;
    title: string;
    completed: boolean;
    dueDate?: string;     // ? = あってもなくてもいい
}
```

### ?. （オプショナルチェーン）
```typescript
category?.color
// categoryがあればcolorを取得、なければundefined
```

### ?? （Null合体演算子）
```typescript
const color = category?.color ?? "gray";
// colorがなければ"gray"を使う
```

### === と ==
```typescript
5 === "5"   // false（型が違う）
5 == "5"    // true（型変換して比較）
// 実務では === を使う
```

### 配列メソッド
```typescript
const todos: Todo[] = [...];

// filter: 条件に合うものだけ残す
todos.filter(t => !t.completed)

// map: 各要素を変換する
todos.map(t => t.title)

// find: 条件に合う最初の1つ（なければundefined）
todos.find(t => t.id === "123")

// forEach: 各要素に処理する（戻り値なし）
todos.forEach(t => console.log(t.title))

// push: 末尾に追加
todos.push(newTodo)
```

### Map（辞書型・高速検索）
```typescript
// 作成
const map = new Map<string, Todo>();

// 登録・取得・存在チェック
map.set("abc", todo);
map.get("abc");          // O(1)で高速
map.has("abc");          // true/false

// 配列から一括作成
const todoMap = new Map(todos.map(t => [t.id, t]));
```

### テンプレートリテラル
```typescript
const name = "太郎";
const msg = `こんにちは、${name}さん`;
// バッククォート ` で囲む。${} の中に変数を入れる
```

### 分割代入
```typescript
// オブジェクトから取り出す
const { title, completed } = todo;

// 配列から取り出す
const [first, second] = [1, 2];
```
