**HTTPメソッドの実例**
- GET: `supabase.from('todos').select('*')` — データ取得
- POST/PUT: `supabase.from(table).upsert(payload)` — 作成or更新
- DELETE: `supabase.from(table).delete().eq('id', data.id)` — 削除

**認証フロー**
`apps/web/src/contexts/AuthContext.tsx`
→ signIn(email, password) → トークン発行 → 以降のリクエストに自動付与

**トークンの保存**

### コードリーディング: TSの文法を読み解く

**`?.`（オプショナルチェーン）**: 値がnull/undefinedかもしれないとき、あれば実行、なければスキップ
**`.find(c => ...)`**: 配列から条件に合う最初の1つを探す（線形探索）
**`c => c.id === todo.categoryId`**: アロー関数。「cのidとtodoのcategoryIdが一致するか」という条件

**再帰でツリー探索するパターン**:
```typescript
const findCat = (cats) => {
    for (const cat of cats) {
        if (cat.id === id) return cat;         // 見つかった
        if (cat.children) findCat(cat.children); // 子も探す（再帰）
    }
};
```
→ 木構造を上から下まで全部たどる。座学の「再帰」と「木構造」がそのまま出てきている
