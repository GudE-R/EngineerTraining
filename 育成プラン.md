# AI時代を生き抜くエンジニア育成プラン

## 目標
- AI時代に自走できるプロフェッショナルエンジニアになる
- 12ヶ月で資格4つ取得（基本情報→応用情報→セキュリティマネジメント→安全確保支援士）
- 新規プロジェクトのアーキテクチャを自力で設計できるレベルに到達

## 現状（2026/03/04 時点）
- **経験**: 未経験・独学で学習中
- **開発**: StudyTodoアプリをReact + React Native + Supabaseで開発中（個人開発）
- **ツール**: Claude Codeを活用した開発フローに慣れつつある
- **資格**: ITパスポート取得済み、基本情報の学習を少し開始
- **弱み**: CS基礎知識、設計力、自走力、実務経験のすべてが課題。英語は苦手
- **強み**: 個人でフルスタックアプリを開発・リリースまで経験している
- **学習環境**: 1日5時間以上確保可能、スタイルはテーマにより使い分け（読む/手を動かす/動画）
- **教材**: Impress参考書（基本情報）、過去問はまだ

## 育成の3本柱
1. **設計力** — 何を作るか・どう構成するかの判断力
2. **問題解決力** — AIに何を聞くか判断できる力
3. **AI協働スキル** — AIを最大限活用する力

※ 3本柱は座学→実践サイクルの中で自然に鍛える（別枠で設けない）

## フェーズ構成

### Phase 1: 基本情報合格（〜Month 3）
- 全25分野の座学+テスト
- StudyTodoでの実践（コード分析・改善）
- 過去問演習で仕上げ → 受験

### Phase 2: 応用情報 + プロダクト開発の並行（Month 4-6）
午前: 応用情報の座学（1-2時間）/ 午後: StudyTodoの開発（3-4時間）

| スキル | やること |
|---|---|
| 設計力 | StudyTodoのコンポーネント設計・状態管理を自力で判断する |
| 問題解決力 | バグを自分で仮説立て→デバッグ→修正する |
| CS基礎 | 応用情報の学習で基本情報の知識を深掘り |
| 実装力 | 新機能を自力で設計→実装→テストまで回す |

応用情報の学習がStudyTodoの設計改善に活き、開発経験が午後問題に活きる相乗効果を狙う。

### Phase 3: 就活 + セキュリティ資格（Month 6-12）
- 実務に入りながら残りの資格を取得（実務経験+資格の相乗効果）

## 12ヶ月後の到達点予測

### 能力レベル
| スキル | 到達レベル | 具体的にできること |
|---|---|---|
| 設計力 | ジュニア〜ミドル | コンポーネント設計、DB設計を自力で判断できる |
| 問題解決力 | ジュニア | デバッグの仮説立て、ボトルネック特定ができる |
| AI協働 | 上位層 | AI活用で実力以上のアウトプットを出せる（大きな差別化） |
| CS基礎 | 応用情報レベル | アルゴリズム、ネットワーク、セキュリティの基礎が体系的 |
| 実装力 | ミドル | フルスタックで個人〜小規模チーム開発を回せる |

### 差別化ポイント
1. 個人でアプリをリリースまで完遂した実績（ポートフォリオとして強い）
2. セキュリティ資格の組み合わせ（未経験者としては異例）
3. AI協働スキル（多くのエンジニアがまだ持っていない）

### 外資ITを目指す場合
| 企業ランク | 具体例 | 12ヶ月後の可能性 | 必要な追加要素 |
|---|---|---|---|
| GAFAM | Google, Amazon, Microsoft, Apple, Meta | ✕ | コーディング面接対策(LeetCode)、英語力、CS学位 or 同等の実力 |
| 外資SaaS | Salesforce, Datadog, Cloudflare | ✕〜△ | 英語力、特定技術領域の深い専門性 |
| 外資コンサル系 | Accenture, Deloitte Digital | △〜○ | 資格は評価される。英語力があれば現実的 |
| 外資IT（日本法人） | AWS Japan, Microsoft Japan（非SWE職） | △ | TAM/SA/CSなど非開発職なら資格+英語で可能性あり |

**外資IT（特にGAFAM SWE）に必要なもの：**
- **LeetCode / アルゴリズム力** — コーディング面接が最大の関門。Medium〜Hard問題を安定して解ける力
- **英語力** — 面接が英語。TOEIC 800+は最低ライン、実際は会話力が必要
- **CS基礎の深さ** — データ構造、アルゴリズム、システムデザインを原理から説明できる
- **実務経験 2-3年** — 新卒以外は基本的に経験者採用

**現実的なルート：**
国内企業で2-3年実務経験 → LeetCode+英語を並行 → 外資に転職（計3-5年）

## 戦略：Month 6で就活開始を推奨
12ヶ月独学を続けるより、6ヶ月で実務に入る方が成長速度は圧倒的に速い。
- Month 1-3: 基本情報合格 + 全分野の概念理解
- Month 4-6: 応用情報 + StudyTodo開発で設計力・実装力を強化 → 就活開始
- Month 6〜: 実務に入りながら残りの資格を取得（実務経験+資格の相乗効果）
- 外資を目指すなら実務経験を積みながらLeetCode+英語を並行
